## Setup Instructions

1. **Install Dependencies**
   Navigate to the project folder and run:
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Features
- **Modern Futuristic Design**: Glassmorphism, circular layouts, and smooth animations.
- **Tech Stack**: React 18, Vite, TailwindCSS, Framer Motion, Lucide Icons.
- **Components**:
  - Circular File/Folder Tiles
  - Floating Sidebar & Navbar
  - Animated Upload Button
  - Theme-aware styles
